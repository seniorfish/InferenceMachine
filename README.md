# 推断机  

![](https://img.shields.io/badge/Javascript-ES2017+-blue)

**推断机** 是一个单页面离线网页程序，能快速解决初中阶段的化学推断题。
程序考虑了初中几乎所有转换式，包括复分解和金属置换，支持添加元素种类限制、元素数量限制等物质条件，支持多种未知物关联方式。
计算结果自动格式化。操作简便，开箱即用。  

[在线DEMO](https://tdj.pages.dev/)
