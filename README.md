# 推断机  
能解决初中阶段的化学推断题。  
打开index.html可以看到具体的使用方法  
### 已知的小bug  
水与二氧化碳既可以生成碳酸，也可以生成葡萄糖和氧气。  
由于映射的特点，转换式条件(a+b=...)在反应物同时为水和二氧化碳时，只考虑生成碳酸。一对一的转换条件不受影响。
